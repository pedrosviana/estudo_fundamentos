#### SDLC summary
> The basics of a SDLC describes the minimum steps we need to have to build a software:
>> Requirement Gathering
>> Planning
>> Design
>> Building / Coding
>> Testing
>> Deploy and maintenance


#### STLC summary
> Describes the inner steps of the testing phase in SDLC
>> Requirement analysis
>> Test planning
>> Design of test cases
>> Set up environment
>> Execution
>> Closure
