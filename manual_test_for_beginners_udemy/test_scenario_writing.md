- **Test scenario:** *Any functionality that can be tested.*
- Also known as **Test condition**
- It's the test activity of the test analisys phase of SDLC
---
### How to write test scenarios?
- Analyse every requirements document available
- Isolate every requirement and identify any possible user actions that need to be tested.
- Write/enumerate test scenarios, create Traceability Matrix to ensure that every requirement is mapped to a test scenario
- If needed, get the scenarios reviewed by a supervisor;
