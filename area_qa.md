> **Passos práticos para organizar**
>> - Falta processo para teste
>> - Falta documentação para teste(pré e pós)
>> - Ta faltando reunião para controle de atividades 
>> - Controle de atividades para estagiários
>> - Definição de métricas para as atividades

---

##### Como organizar métricas de teste?
- O plano de teste fala de indicadores e métricas para avaliar o teste que está sendo desempenhado.
- Bugs por estoria
- Tempo para correção de bug
- Tempo execução de suite de teste
